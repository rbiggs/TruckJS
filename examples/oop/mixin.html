Truck Object Mixin

Truck's $.mixin method lets you add one object's properties to another.

  • Original object:

    var sam = {
  name: 'Sam',
  job: 'developer',
  hobby: 'twiddling thumbs',
  address: '1234 Lane, SomeWhereVille'
}

  • New object to mixin:

    var newProps = {
  job: 'cook',
  address: '000 DeadEnd, NoPlace',
  age: 1000
};

  • Mixin newProps into sam:

    $.mixin(newProps, sam);

  • Resulting object:

    var sam = {
  name: 'Sam',
  job: 'developer',
  hobby: 'twiddling thumbs',
  address: '1234 Lane, SomeWhereVille'
  age: 1000
}
